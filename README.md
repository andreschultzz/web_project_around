# 🌄 Around The U.S.

Um site de portfólio de viagens que apresenta fotos de paisagens deslumbrantes dos Estados Unidos. Criado para amantes da natureza, o site combina uma interface moderna e escura com belas imagens de parques nacionais e locais icônicos.

## ✨ Funcionalidades

- Formulário para alteração das informações de perfil.
- Layout responsivo.
- Botões curtir interativos.

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
